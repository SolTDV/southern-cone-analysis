# Southern Cone Economic & Demographic Analysis

Midterm project for MATH105 - Introduction to Statistics

## Overview

This project examines the long-term demographic and economic trends in Uruguay, Chile and Argentina (The Southern Cone) through the Gapminder dataset

## What I did
- Cleaned and joined Gapminder data for the three countries, checking for NA values and outliers.
- Visualized trends in life expectancy, fertility rate, and population growth under the form of barplots, scatter plots and line plots.
- Analyzed pairwise relationships
## Tools
R, tidyverse, ggplot2
## Files
"Report.Rmd": the source code
"Report.HTML": rendered report
